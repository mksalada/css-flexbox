CSS Flexbox Alignment: place-content Property

Notice: Except for the center center value of the place-content property, everything will not display for both in row and column. 

.content {
    display: flex;
    flex-wrap: wrap;
}


.content1 {
    place-content: flex-start flex-start;
}
/* default */
/*
    desktop/tablet screens:
        space-between is effective on rows

    mobile screens:
        space-between is effective on columns
*/

.item1
.item2 



.content2 {
    place-content: flex-end flex-end;
}
/*
    desktop/tablet screens:
        space-between is effective on rows

    mobile screens:
        space-between is effective on columns
*/

.item1
.item2 



.content3 {
    place-content: center center;
}

.item1
.item2
